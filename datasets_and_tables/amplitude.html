Amplitude¶

Amplitude is used to track key user events in the Benefits application. Data from the Amplitude API is ingested into the ingest_amplitude_raw_dev and ingest_amplitude_raw_prod bucket, in a folder called /benefits/. The view makes the data available to users in Metabase.

The DAG loads events from Amplitude API to Google Cloud Storage as flattened JSON, moves JSONL to BigQuery table and moves BigQuery table to a view.
